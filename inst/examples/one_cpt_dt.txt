********** MODEL NAME

Simple One Compartment Dosing Model

********** MODEL NOTES

A simple one compartment dosing model where
the doses must be defined as a data frame in
the 'dosingTable' argument of the simulate
function.

********** MODEL STATES

d/dt(y) = -theta * y + INPUT1

y(0) = 0

********** MODEL PARAMETERS

theta = 0.6
